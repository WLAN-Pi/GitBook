# What's New In Version 2?

The WLAN Pi V2 image was released in October 2020. The image has been rebuilt from the ground up to provide an improved software installation and update process. It's toolset has been revamped, it has many usability improvements and hardware support has been expanded.

## Version 2.0.0 Announcement Notes <a id="version-200-announcement-notes"></a>

You can find the official v2.0.0 announcement [here](https://wlan-pi.github.io/wlanpi-documentation/v2/v2_announcement/). It details the main feature of the [new v2.0.0 release](https://wlan-pi.github.io/wlanpi-documentation/v2/v2_announcement/).

## Version 2.0.0 Video & Slides <a id="version-200-video-slides"></a>

The WLAN Pi development team presented a collaborative video for the online CWNP 2020 WTF conference that details the main features and improvements of the version 2 image. [Links to the video and slide deck](https://wlan-pi.github.io/wlanpi-documentation/v2/v2_video/) are provided on this [page](https://wlan-pi.github.io/wlanpi-documentation/v2/v2_video/).

## Getting the version 2 Image <a id="getting-the-version-2-image"></a>

If you'd like to get hold of the latest version 2 image for your WLAN Pi, you can download the software image from here: [WLAN Pi Images](https://github.com/WLAN-Pi/releases/releases) \(look for the assets section that contains the image - the file name will be something like: `wlanpi-neo2-v2.0.0.img.gz`\). Note that bugfixes and improvement are being continually added, so please ensure you download the latest 2.x.x image version.

To burn your image on to an SD card, the information provided in this video is still good, even though it was created for a v1.x.x image: [How to Burn a New Image on Your WLAN Pi \(WiFiNigel\)](https://youtu.be/sD4WlNyyWDs)
