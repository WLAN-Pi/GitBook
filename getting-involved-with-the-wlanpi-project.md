# Getting involved with the WLANPi project

The WLANPi project is a community effort and we always welcome volunteers to contribute to our efforts. There are opportunities for coding, testing \(no coding skills required\) and documentation/web site support.

To find out the different areas where you can help out, take a look at the following pages for more information:

* [Coding for the WLAN Pi](https://wlan-pi.github.io/wlanpi-documentation/get_involved/get_involved_coding/)
* [Contributing to development for non-coders](https://wlan-pi.github.io/wlanpi-documentation/get_involved/get_involved_non_coding/)
* [Submitting bug reports](https://wlan-pi.github.io/wlanpi-documentation/get_involved/get_involved_bugs/)
* [Submitting feature/package suggestions](https://wlan-pi.github.io/wlanpi-documentation/get_involved/get_involved_requests/)

If you'd like to be involved, please contact [Jerry Olla](https://twitter.com/jolla) or [Nigel Bowden](https://twitter.com/wifinigel) on Twitter or track them down via any well known search engine.
